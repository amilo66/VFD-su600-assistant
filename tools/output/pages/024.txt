F3.00

The PID function setting

LED single bit: PID adjusting property
0: Invalid

1: Positive effect
When the feedback signal is greater than the PID,
the inverter output frequency is required to decrease
(i. e., reduce the feedback signal).

2: Negative effect

When the feedback signal is greater than the PID, the
inverter output frequency is required to increase

(i. e., reduce the feedback signal).

LED ten place: PID to the quantitative input channel
0: The keyboard potentiometer

PID to quantification is given by the potentiator on
the operator panel.

1: Number given

The PID ration is given by the number and set by the
function code F3.01.

2: Pressure given (MPa, Kg)

Set the given pressure by setting F3.01, F3.18.
